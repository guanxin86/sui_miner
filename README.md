## 运行命令
### 只挖FOMO或Meta
```
cd ./sui_mete_miner
node mine.js --meta --chain=mainnet --phrase="secretphrase"
```
```
cd ./sui_mete_miner
node mine.js --fomo --chain=mainnet --phrase="secretphrase"
```
### 同时挖FOMO或Meta
```
node mine.js --fomo --meta --chain=mainnet --phrase="secretphrase"
```
